## Q: 单向链表的 二元分割
+ T(n) = O(n)
+ S(n) = O(1)
+ 即快速排序可以应用于 单向链表
+ 方法:
    1. 两个单向链表 头指针 pl pr
    2. 小于 pivot的添加到 pl链表后面
    3. 大于 pivot的添加到 pr链表后面
    4. 注意pl的尾要接上pr的头
